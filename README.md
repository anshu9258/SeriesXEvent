SeriesX Events 🎉

SeriesX Events is a modern event management and booking platform built using Spring Boot, MySQL, HTML, CSS, and JavaScript. The platform allows users to explore event services, submit inquiries, and connect directly through WhatsApp.

🚀 Features
Professional event management website
Responsive modern UI
Contact form with database integration
WhatsApp direct redirection
Newsletter subscription
MySQL database integration
REST API backend
Railway cloud deployment
GitHub integration

⚙️ Setup Instructions
1️⃣ Clone Repository
git clone https://github.com/anshu9258/SeriesXEvent.git
2️⃣ Open Project

Open the project in:

IntelliJ IDEA
VS Code
Eclipse
3️⃣ Configure Database

Update application.properties:

spring.datasource.url=jdbc:mysql://localhost:3306/seriesxevents
spring.datasource.username=root
spring.datasource.password=yourpassword
4️⃣ Run Project
mvn spring-boot:run
